Clockwork - Kom ihåg när saker hände i dina Homey-flöden

Spåra händelsetidsstämplar och använd dem för att skapa intelligenta flöden som förstår timing och sekvens.

VIKTIGA FUNKTIONER:
• Spara tidsstämplar för valfri händelse med anpassade identifierare
• Kolla hur länge sedan händelser inträffade (mindre än, mer än eller mellan tidsintervall)
• Jämför händelsetajming och ordning (före, efter eller samtidigt)
• Kolla om händelser någonsin har hänt eller aldrig har hänt
• All data bevaras även efter omstart av appen - ingen dataförlust
• Smart autofyll visar dina befintliga händelser med tidsstämplar
• Utlös flöden när händelser sparas

PERFEKT FÖR:
• Identifiera rörelseriktning (t.ex. gå upp eller nerför trappor)
• Bygga "välkommen hem"-flöden som endast utlöses efter frånvaro
• Skapa sekvenser som beror på händelsetajming
• Övervaka användningsmönster för apparater
• Förhindra dubbelnotifikationer inom tidsramar

HUR DET FUNGERAR:
1. Använd åtgärden "Spara att [event_id] hände" för att registrera händelser
2. Använd villkorskort för att kontrollera tajming (t.ex. "door_opened hände för mindre än 10 sekunder sedan")
3. Bygg komplexa automationer baserade på händelsesequencer och tajming

EXEMPEL - Smart Trappbelysning:
Lagra dörröppningar → Kontrollera om rörelsesensor aktiverats inom 10 sekunder → Bestäm om användaren går upp eller ner → Tänd/släck ljus därefter

Alla tidsstämplar sparas lokalt på din Homey för fullständig integritet.
