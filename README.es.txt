Clockwork - Recuerda cuándo ocurrieron cosas en tus flujos de Homey

Accede a las marcas de tiempo de eventos y úsalas para crear flujos inteligentes que entiendan el momento y la secuencia.

FUNCIONES PRINCIPALES:
• Almacena marcas de tiempo de cualquier evento con identificadores personalizados
• Averigua cuánto tiempo ha pasado desde que ocurrieron eventos (menos de, más de, o entre intervalos de tiempo)
• Compara el momento y el orden de eventos (antes, después, o simultáneo)
• Comprueba si los eventos han ocurrido alguna vez o nunca
• Todos los datos se mantienen al reiniciar la aplicación - sin pérdida de datos
• Autocompletar inteligente muestra tus eventos existentes con marcas de tiempo
• Dispara flujos cuando se almacenan eventos

PERFECTO PARA:
• Detectar la dirección de movimiento (por ejemplo, subiendo o bajando escaleras)
• Crear flujos de "bienvenida a casa" que se activen solo tras ausencias
• Crear secuencias que dependan del momento de eventos
• Monitorear patrones de uso de electrodomésticos
• Prevenir notificaciones duplicadas dentro de ventanas de tiempo

CÓMO FUNCIONA:
1. Usa la acción "Almacenar que ocurrió [event_id]" para registrar eventos
2. Usa tarjetas de condición para verificar tiempos (por ejemplo, "puerta_abierta ocurrió hace menos de 10 segundos")
3. Construye automatizaciones complejas basadas en secuencias y tiempos de eventos

EJEMPLO - Iluminación Inteligente de Escaleras:
Almacenar apertura de puerta → Verificar si el sensor de movimiento se activó en 10 segundos → Determinar si el usuario sube o baja → Encender/apagar luces según corresponda

Todas las marcas de tiempo se almacenan localmente en tu Homey para completa privacidad.
