Clockwork - 홈이 플로우에서 발생한 일을 기억하세요

이벤트 타임스탬프를 추적하고 이를 사용하여 시퀀스 및 타이밍을 이해하는 지능형 플로우를 만드세요.

주요 기능:
• 사용자 지정 식별자와 함께 모든 이벤트의 타임스탬프 저장
• 이벤트가 발생한 지 얼마나 지났는지 확인 (보다 작음, 보다 큼 또는 시간 범위 사이)
• 이벤트 타이밍 및 순서 비교 (이전, 이후, 동시에)
• 이벤트가 발생했는지 또는 발생하지 않았는지 확인
• 앱 재시작 시에도 모든 데이터 유지 - 데이터 손실 없음
• 스마트 자동 완성으로 기존 이벤트의 타임스탬프를 보여줌
• 이벤트가 저장될 때 플로우 트리거

완벽한 사용 사례:
• 이동 방향 감지 (예: 계단 오르내리기)
• 부재 후에만 트리거되는 "환영 귀가" 플로우 생성
• 이벤트 타이밍에 의존하는 시퀀스 생성
• 가전제품 사용 패턴 모니터링
• 일정 시간 내 중복 알림 방지

사용 방법:
1. "이벤트_id가 발생했다" 액션을 사용하여 이벤트 기록
2. 조건 카드를 사용하여 타이밍 확인 (예: "door_opened가 10초 이내에 발생했다")
3. 이벤트 시퀀스와 타이밍을 기반으로 복잡한 자동화 구축

예시 - 스마트 계단 조명:
문 열림 저장 → 10초 이내에 모션 센서가 트리거되었는지 확인 → 사용자가 올라가는지 내려가는지 판단 → 이에 따라 조명 켜기/끄기

모든 타임스탬프는 완전한 개인정보 보호를 위해 Homey에 로컬로 저장됩니다.
